# CampusEats Frontend Framework

This is the basic PR2 frontend framework for CampusEats.

## Tech Stack

- Vue 3
- Vue Router
- Pinia
- Vite
- Mock JSON data

## Project Setup

To run the frontend:

```bash
cd frontend
npm install
npm run dev
```
